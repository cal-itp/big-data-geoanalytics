# Project Analysis

Looking into [CTC commission meeting agenda items](https://catc.ca.gov/meetings-events/commission-meetings-archive/meeting-archives-2025) from 2021 to 2026. We scraped the project data that relates to project cost amendments & project time extensions (item numbers 2.5() and 2.8() respectively)

[project_analysis_report.ipynb](./project_analysis_report.ipynb) includes a rough draft report of the project analysis data by district, county, and item type.
