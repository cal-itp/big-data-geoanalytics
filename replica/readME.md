# Replica Functions

This folder contains some utils and other helpful functions and analyses that others can utilize when working with Replica.


Folders: 
* [LinkUS_locations_selections](LinkUS_locations_selections): This folder contains functions that combine any geography locations selected in the Replica Platform in the Saved Selection section. Replica's platform does not allow for the predefined selections to be combined. This also has the first round of trip analysis, done for LinkUS, to determine what kinds of trips are happening between the stations.
* [transit_od_analysis](transit_od_analysis): This folder contains the utils and analyses looking into the different types of trips occuring between origins and destinations for one tract or area. Specifically, an analysis of trips to and from Cal Poly Humboldt
* [StateWideTravelMarkets](StateWideTravelMarkets): This folder contains the analysis for trip analyses between POIs across California, using trip data from Replica.
